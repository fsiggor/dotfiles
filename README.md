# dotfiles

macOS Dev Environment Setup — Alacritty, Chromium, Bitwarden, Docker, mise, Neovim, tmux, OpenCode, Antigravity, Oh My Zsh, Go, Rust, Node.js e Python.

## Instalacao remota

```bash
curl -fsSL https://raw.githubusercontent.com/fsiggor/dotfiles/main/install.sh | bash
```

## Instalacao local

```bash
git clone https://github.com/fsiggor/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```
